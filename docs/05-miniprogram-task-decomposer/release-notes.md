# 微步 ACTION · 版本日志

> 本页记录微步 ACTION 的功能变化、已知限制和后续计划。

---

## v1.0.0（2026-04-20）

### 新增

* 上线任务输入与 AI 拆解核心流程；
* 支持将用户输入的大目标拆解为 5 个小步骤；
* 支持五步卡片式展示；
* 支持 checkbox 勾选完成项；
* 支持进度条实时更新；
* 完成全部步骤后，发放积分奖励；
* 新增任务与成就页面，展示进行中任务和历史完成记录；
* 新增基础用户文档、PRD、API 参考文档和版本日志。

### 改进

* 优化任务输入提示，引导用户提供更具体的目标、时间范围和当前状态；
* 统一任务拆解结果结构，便于前端展示步骤卡片；
* 增加常见错误场景说明，包括输入过短、输入过长、网络异常和 AI 服务不可用。

### 已知限制

* 当前版本强依赖网络环境，离线不可用；
* 单次拆解结果暂不支持手动编辑；
* 拆解结果质量依赖用户输入的具体程度；
* 当前版本仅支持固定 5 步拆解；
* 勋章和积分玩法仍较轻量，后续会继续补充。

---

## v1.1.0（计划中）

### 计划新增

* 支持编辑单个步骤；
* 支持对拆解结果进行“不准确 / 太难 / 太泛”反馈；
* 支持更多勋章样式；
* 支持不同任务类型的拆解模板；
* 优化任务历史记录展示。

### 计划改进

* 优化 AI Prompt，让拆解结果更贴近“15 分钟内可完成”的标准；
* 增加输入示例，引导用户写出更好的任务描述；
* 优化错误提示语气，减少用户挫败感。
