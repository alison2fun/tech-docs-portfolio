# Changelog

这个页面记录作品集站点本身的结构、内容和交互更新。项目文档质量自动化流水线的更新记录仍保留在 [更新记录](changelog.md)。

## v2.1.0 - 阅读路径与语气调整

发布日期：2026-07-10

### 主要改动

* 将思考与经验、案例复盘卡片的入口合并到标题，移除重复按钮；
* 把写作样稿从长表格改为按读者任务分类的文档索引；
* 重写首页、关于我和总览页导语，让页面从真实问题和阅读摩擦开始；
* 将项目详情页的“证明能力”段落改为文档结构与写作取舍说明；
* 扩充优秀文档案例复盘，并减少跨页面重复的模板标题；
* 统一底部导航的尺寸、边界和响应式对齐。

## v2.0.0 - 职业化优化

发布日期：2026-07-07

### 主要改动

* 重构首页文案与入口权重，突出 Docs-as-Code 技术文档系统定位；
* 将“文稿”调整为“思考与经验”，强化方法论和职业判断表达；
* 压缩并重构简历页，按背景、挑战、解决方案和产出组织项目；
* 为简历页增加 PDF 下载入口，并同步生成新版 PDF 简历；
* 为作品页增加 Context 字段，统一呈现解决的问题、目标读者和交付物形式；
* 为作品卡片增加 200 × 150 缩略图，提升作品证据密度；
* 将案例复盘调整为过程记录结构，补充技术选型、执行过程和最终作品链接；
* 增加英文未完成提示和页面底部阅读引导；
* 统一缩略图、按钮、底部导航在日间和夜间模式下的样式。

## v1.0.0 - 作品集结构改版

* 从“简历堆叠”改为“技术文档作品集 + 求职叙事”的混合结构；
* 增加 About、Portfolio、Case Study、Resume 等清晰入口；
* 修复发布路径中的 `.md` 链接问题；
* 增加夜间模式和首页居中个人入口。
