### 硬件数据手册重构

**背景**：模拟将传统芯片 PDF 数据手册改写为开发者友好的 Web 文档。

**核心挑战**：寄存器、SPI 时序、bit 位和代码配置分散在 PDF 中，检索和复用成本高。

**解决方案**：

* 用 Mermaid 重画 SPI 读取流程；
* 将寄存器地址、默认值、访问权限和 bit 定义整理为结构化表格；
* 补充 C 语言初始化示例，并解释代码与寄存器字段的对应关系。

**产出结果**：[硬件数据手册重构](https://alison2fun.github.io/tech-docs-portfolio/02-hardware/)。

### IoT 接口集成指南

**背景**：模拟开发者通过 RESTful API 获取光感传感器实时数据的接入场景。

**核心挑战**：API 文档容易停留在字段堆叠，缺少认证链路、可复制示例和排错路径。

**解决方案**：

* 拆分认证流程、Endpoint、Header、Path / Query 参数和响应字段；
* 提供 curl 请求示例、JSON 响应示例和错误码表；
* 用 Mermaid 展示 OAuth 风格调用链路，并补充调试建议。

**产出结果**：[IoT 接口集成指南](https://alison2fun.github.io/tech-docs-portfolio/03-api/)。
